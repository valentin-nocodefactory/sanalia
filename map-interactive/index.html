    À lire avant toute citation

    Les données présentées comptent des signalements, pas des populations biologiques. Une commune avec beaucoup de signalements n'est pas nécessairement plus infestée — elle est aussi plus mobilisée numériquement (sur-représentation des grandes villes équipées d'une plateforme citoyenne).

    Sanalia est une entreprise de lutte contre les nuisibles : nous avons donc intérêt à ce que le sujet soit visible. Pour limiter ce biais, le code du pipeline est ouvert, les données publiques sont séparées de nos propres interventions, et nous nous engageons à ne jamais modifier les sources publiques.

    Questions fréquentes

    À quelle fréquence les données sont-elles mises à jour ?
    Les sources avec API publique (Paris Open Data, Atlasanté ambroisie) sont rafraîchies quotidiennement. Les datasets statiques (ANSES moustique tigre, INPN frelon, étude ANSES punaises de lit) sont mis à jour à chaque publication officielle. Une consolidation complète est effectuée le 1er de chaque mois.
    Pourquoi les punaises de lit ne sont-elles pas affichées sur la carte ?
    La seule source nationale rigoureuse (étude ANSES 2023) ne fournit qu'une granularité régionale, pas commune par commune. Représenter ces données comme des points serait trompeur. Nous les présentons donc dans un bloc dédié, par région, avec les taux d'infestation officiels.
    Comment est calculé le « nuisible dominant » d'une commune ?
    C'est l'espèce qui représente le plus grand nombre de signalements dans la commune sur les 12 derniers mois. Nous montrons la couleur du nuisible dominant comme indicateur visuel principal, mais le détail complet par espèce est disponible en cliquant sur la commune.
    Puis-je utiliser cette carte dans un article de presse ?
    Oui, gratuitement. La carte est sous licence Creative Commons BY 4.0, les données brutes sous ODbL. Mention recommandée : « D'après l'Observatoire Sanalia, croisant les données publiques de l'ANSES, du MNHN et des collectivités ». Pour un angle dédié ou une note de contexte, écrivez à presse@sanalia.fr.
    Pourquoi Paris ressort-il autant ?
    Paris dispose depuis 2012 de l'application Dans Ma Rue, qui collecte massivement les signalements citoyens. La majorité des autres communes françaises n'ont pas d'équivalent — Paris est donc mécaniquement sur-représentée. Cela ne signifie pas que Paris est plus infestée que d'autres villes : la mobilisation numérique y est simplement plus structurée.
    Quelles sont les limites de cet observatoire ?
    Trois limites principales : (1) biais déclaratif des signalements citoyens, (2) couverture géographique inégale entre les communes, (3) sources hétérogènes en méthodologie (un signalement Dans Ma Rue ≠ une intervention SMASH ≠ une commune classée colonisée). Les comparaisons inter-sources sont à manier avec prudence. La page méthodologie détaille chaque biais.
